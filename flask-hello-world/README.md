# Flask Hello World Project

This is a simple Flask application that displays "Hello World" on the frontend.

## Project Structure

```
flask-hello-world
├── app.py
├── templates
│   └── index.html
├── static
│   └── css
│       └── styles.css
├── requirements.txt
├── .gitignore
└── README.md
```

## Requirements

To run this application, you need to have Python and Flask installed. You can install the required packages using the following command:

```
pip install -r requirements.txt
```

## Running the Application

1. Navigate to the project directory:
   ```
   cd flask-hello-world
   ```

2. Run the application:
   ```
   python app.py
   ```

3. Open your web browser and go to `http://127.0.0.1:5000` to see the "Hello World" message.
